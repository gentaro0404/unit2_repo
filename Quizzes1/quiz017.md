## Create a function that receives as input a stering and reaturns the string ciphered with shift 13.

## :program
```.py
import math

number = int(input("Enter a number: "))
x = -12


while -12<=x<=15:
    power = math.pow(10, x)
    print(f"{number} * 10^{x} = {number*power}")
    x += 1

```
